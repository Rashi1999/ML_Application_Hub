Objective of Machine Learning Hub

Machine learning is a branch of artificial intelligence (AI) and computer science which focuses on the use of data and algorithms to imitate the way that humans learn, gradually improving its accuracy. Machine Learning is one of the hottest research topics in computer science and engineering which is applicable in many disciplines. It provides a collection of algorithms, methods and tools capable of embodying some kind of intelligence to machines. This project aims to club together different ML applications under one roof to demonstrate the wide scope of this domain in medical diagnosis, email filtering, recommendation systems or future price prediction of commodities, chatbots, natural language processing among many others. It uses Flask for Frontend Development and Python at the backend for Machine Learning models.

Code Visibility

The project has a view code module which outlines the detailed step by step process of model building. This module provides a learning pathway for the individuals keen to venture in the world of data science and learn about the ML algorithms along with their applications.

Visualisation Feature

The visualisation feature demonstrates graphically how different independent features of the dataset affect the dependent target value and represents the correlation among the attributes.
